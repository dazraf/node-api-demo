Service API

  • GET wallets

    This returns an object mapping the "wallet" id to a wallet structure.

    Example:

    { "w1": { "id":"w1", "name":"Oliver", "balance":100000, "account":"11223344", "sortCode":"112233" } }
  • GET wallets/:walletId

    This the "wallet" object for the given walletId.

    Example:

    { "id":"w1", "name":"Oliver", "balance":100000, "account":"11223344", "sortCode":"112233" }
